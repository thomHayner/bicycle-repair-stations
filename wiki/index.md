# Index

A content-oriented catalog of every page in this wiki. Read this first when answering a query — it's faster than scanning `pages/` directly, and it's maintained as a first-class artifact by every ingest.

The categories below are seeds. Rename, merge, or split them as the wiki's content dictates — the goal is a catalog that helps you find things, not adherence to this starting taxonomy.
